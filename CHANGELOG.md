# Change Log

All notable changes to the "mcp-screenshot" extension will be documented in this file.

## [0.0.1] - 2024-01-15

### Added
- Initial release
- Full screen capture command
- Window capture command with window selection
- Region capture command with coordinate input
- Display listing command
- Window listing command
- Configuration settings for format, quality, and save directory
- PII masking support
- Multi-monitor support
- Auto-save functionality
- MCP server integration
- Output channel for debugging

### Features
- Cross-platform support (Linux, macOS, Windows)
- Multiple image formats (PNG, JPEG, WebP, BMP)
- Configurable compression quality
- Automatic directory creation
- Privacy protection with PII masking
- Security controls and rate limiting

### Documentation
- README with usage examples
- Configuration guide
- Troubleshooting section
